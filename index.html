Keyboard Shortcuts

Click on a box to select it, then press:

  • Esc to deselect the box
  • C to clone the box
  • R to resize the box
  • I to convert the box to an image
  • X to delete the box
  • B to send the box backward
  • F to bring the box forward
  • Up to move the box up
  • Down to move the box down
  • Left to move the left
  • Right to move the right
  • S save the current wireframe
  • L load a saved wireframe
  • D delete a saved wireframe
  • W wipe the screen clean
